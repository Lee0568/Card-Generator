# 卡牌生成器功能与优化清单（Roadmap）

## 必做（高优先级）
- 扩展成本为动态列表：可增删、排序、选择图标库
- 本地草稿保存与加载：自动保存到 `localStorage`，支持“另存为”、“打开”
- 主题系统增强：自定义色板、暗色模式、卡框样式开关
- 导出增强：透明背景、倍率（1x/2x/3x）、边距/出血控制
- 版式细化：标签位置/大小、标题行高与溢出省略、描述自动换行与强调数字
- 校验提示面板：展示错误列表与修复建议（靠右上角弹出）

## 重要（中优先级）
- 批量生成：CSV/JSON 导入 → 生成多张卡片并一键打包 ZIP
- 模板管理：保存/加载模板，快速切换不同卡牌类型（单位/科技/事件等）
- 图标来源：内置矢量图标库（Heroicons/Material），支持搜索与颜色统一
- 快捷键与编辑体验：撤销/重做（Ctrl+Z/Y）、复制卡片、清空表单、一键重置
- 多语言：界面 i18n 与卡片文案多语言输入（中/英切换）

## 进阶（可选）
- 排版网格与吸附：拖拽微调、对齐参考线、组件锁定
- 批量版面：A4/网格导出预览，适合打印和试玩集卡
- PWA 离线支持：可离线编辑、缓存资源、安装到桌面
- 权重与平衡建议：根据“能量/成本/时代”给出强度预算提示
- 资源管理：图片上传目录与压缩、SVG 颜色统一、版权标注区

## 技术与性能
- 模块化与打包：拆分 JS 模块、使用轻量打包器（可选）
- 性能优化：懒加载图标库、图片压缩、预渲染常用模板
- 无障碍与可访问性：对比度检查、键盘导航、ARIA 标签

---

### 里程碑建议
1. M1（基础完善）：动态成本、保存/加载、主题增强、导出增强、校验面板
2. M2（批量与模板）：批量生成、模板管理、图标库、快捷键
3. M3（进阶体验）：网格/吸附、打印版面、PWA、平衡建议

### 备注
- 当前已支持：实时预览、PNG导出、复制JSON、基础主题、Schema校验。
- 若你提供目标风格样例，我会将卡片细节（字体、阴影、边距、色板）对齐到统一标准。
